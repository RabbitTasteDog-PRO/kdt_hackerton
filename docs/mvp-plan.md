# MVP Plan

## Goal

쉬었음 청년과 노인이 자신의 상황에 맞는 기회를 찾고, 바로 실행할 수 있는 다음 행동을 얻도록 돕습니다.

## User Input

MVP에서는 긴 회원가입 대신 간단한 입력 폼으로 시작합니다.

- 사용자 유형
- 나이대
- 거주 지역
- 현재 상태
- 관심 분야
- 사용 가능한 시간
- 필요한 도움

## AI Diagnosis

입력값을 바탕으로 AI가 사용자의 현재 상황을 짧게 요약합니다.

예시:

- 취업 전환이 필요한 쉬었음 청년
- 지역 기반 복지와 커뮤니티 연결이 필요한 노인
- 단기 교육과 상담 지원을 먼저 탐색해야 하는 사용자

## Recommendation Categories

### 쉬었음 청년

- 취업 지원 프로그램
- 직무 교육
- 자격증 과정
- 심리 상담
- 청년 정부지원
- 멘토링
- 커뮤니티

### 노인

- 노인 일자리
- 복지 서비스
- 건강 관리 프로그램
- 지역 커뮤니티
- 디지털 교육
- 상담 서비스
- 여가 프로그램

## MVP Screen Flow

```text
온보딩
↓
상황 입력
↓
AI 진단 결과
↓
추천 기회 목록
↓
추천 상세
↓
실행 체크리스트
↓
진행 기록
```

## Recommendation Card

각 추천 항목은 아래 정보를 포함합니다.

- 제목
- 대상 사용자
- 추천 이유
- 기대 효과
- 실행 난이도
- 예상 소요 시간
- 다음 행동

## Example Output

```text
추천: 청년도전지원사업
이유: 최근 구직 활동을 쉬고 있고, 다시 사회활동을 시작하려는 사용자에게 적합합니다.
다음 행동: 거주 지역 청년센터에서 모집 여부를 확인하세요.
체크리스트:
1. 신청 자격 확인
2. 지역 운영기관 확인
3. 상담 신청
```

## Demo Scope

해커톤 데모에서는 실제 외부 API 연동보다 사용자 입력, AI 진단, 추천 결과, 체크리스트 생성의 흐름을 완성도 있게 보여주는 데 집중합니다.

## Future Expansion

- 실제 공공데이터 API 연동
- 지역 기반 추천 고도화
- 사용자 진행 기록 기반 재추천
- 보호자 또는 상담사 연동
- 경력단절 여성, 장애인, 청소년 등 대상 확장
