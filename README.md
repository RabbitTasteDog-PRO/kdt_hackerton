# AI Opportunity OS

AI와 함께, 인간의 가능성을 확장하는 기회 추천 플랫폼입니다.

## Overview

AI Opportunity OS는 사용자의 현재 상황을 분석하고, 다음 단계로 이동하는 데 필요한 기회를 연결하는 서비스입니다. 교육, 취업, 부업, 정부지원, 멘토링, 지역 서비스, 커뮤니티처럼 흩어져 있는 정보를 개인의 상황에 맞게 추천합니다.

해커톤 MVP는 범위를 넓게 잡기보다 아래 두 사용자군에 집중합니다.

- 쉬었음 청년
- 노인

## Problem

정보는 많지만 개인에게 맞는 기회는 잘 연결되지 않습니다.

특히 쉬었음 청년과 노인은 필요한 지원 제도, 교육, 일자리, 지역 서비스를 찾기 어렵고, 찾더라도 무엇부터 실행해야 할지 판단하기 어렵습니다.

## Solution

사용자의 나이, 지역, 관심사, 현재 상태, 시간 여유, 필요한 도움을 입력받고 AI가 현재 상황을 진단합니다. 이후 사용자가 바로 실행할 수 있는 기회와 단계별 실행 계획을 추천합니다.

```text
사용자 정보 입력
↓
AI 현재 상태 진단
↓
맞춤 기회 추천
↓
실행 계획 생성
↓
실행 결과 기록
↓
다음 기회 재추천
```

## MVP Features

- 사용자 상황 입력
- AI 기반 현재 상태 요약
- 사용자 유형별 기회 추천
- 추천 이유 제공
- 실행 체크리스트 생성
- 진행 상태 기록

## Target Users

### 쉬었음 청년

- 다시 사회활동을 시작하고 싶은 청년
- 취업, 교육, 자격증, 상담, 정부지원 정보를 찾는 사용자
- 무엇부터 해야 할지 막막한 사용자

### 노인

- 은퇴 이후 일자리, 복지, 건강, 커뮤니티 정보를 찾는 사용자
- 지역 기반 지원 서비스를 쉽게 찾고 싶은 사용자
- 디지털 정보 탐색이 어려운 사용자

## Vision

이번 해커톤에서는 쉬었음 청년과 노인을 대상으로 MVP를 개발합니다. 하지만 동일한 구조를 적용하면 경력단절 여성, 장애인, 청소년, 은퇴자, 지방 거주자 등 다양한 계층으로 확장할 수 있습니다.

AI Opportunity OS의 목표는 모든 사람에게 다음 기회를 연결하는 것입니다.
